*******************************************************************************
This command takes as input "year, month, day" in Gregorian calendar and 
returns the corresponding dates in Solar Hijri calendar.

** Note 1: 
Gregorian calendar inputs must be provided in this order: year month day

** Note 2:
All three input 'date' variables must be provided in 'numeric' format.
*******************************************************************************



*******************************************************************************
To run the test file for "greg2sol" command follow these steps:
1. Move "greg2sol.ado" to your personal adopath directory (or you may use 
	"main.do" to specify your personal settings).
2. Change the global variable ${root} to your "greg2sol" folder directory.
3. Run "test_greg2sol.do".
